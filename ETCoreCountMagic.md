# <et:core:count\_magic> #

## Namespace ##
core

## Type ##
Single

## Description ##
Outputs the number of magic parameters passed to current page request.

## Attributes ##
None

## Examples ##
```
	<et:core:count_magic />
```

Since the core namespace is always available, this tag may be abbreviated by omitting the namespace.

```
	<et:count_magic />
```
